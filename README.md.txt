# Proyecto de Python

Este proyecto implementa varios programas que demuestran el uso de conceptos fundamentales de Python.

## Contenido

1. **temperature_converter.py**
   - Conversor de unidades de temperatura (Celsius a Fahrenheit y viceversa).

2. **user_form.py**
   - Formulario en consola que solicita información al usuario y la almacena en variables.

3. **number_sign.py**
   - Programa que determina si un número es positivo, negativo o cero.

4. **multiplication_table.py**
   - Generador de tablas de multiplicar y calculadora de factoriales.

5. **contacts.py**
   - Sistema de agenda de contactos utilizando diccionarios.

6. **area_calculator.py**
   - Calculadora de áreas de figuras geométricas utilizando funciones.

## Instalación

Este proyecto no requiere dependencias adicionales. Puedes ejecutar los scripts directamente desde la terminal.
