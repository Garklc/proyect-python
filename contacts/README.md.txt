# Sistema de Agenda de Contactos

Este programa utiliza diccionarios para almacenar y gestionar una agenda de contactos.

## Características

- Agregar nuevos contactos con nombre, teléfono y email
- Visualizar todos los contactos almacenados
- Buscar contactos por nombre
- Almacenamiento en memoria usando diccionarios

## Uso

Ejecuta el programa con:
```bash
python contacts.py
