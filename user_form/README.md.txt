# Formulario en Consola

Este programa solicita información al usuario y la almacena en variables adecuadas, demostrando el uso de diferentes tipos de datos en Python.

## Características

- Captura de nombre (string)
- Captura de edad (integer)
- Captura de estado de estudiante (boolean)
- Captura de altura (float)
- Captura de peso (float)

## Uso

Ejecuta el programa con:
```bash
python user_form.py
