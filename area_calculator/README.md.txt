# Calculadora de Áreas de Figuras Geométricas

Este programa utiliza funciones para modularizar el cálculo de áreas de diferentes figuras geométricas.

## Características

- Calcula el área de un círculo usando la fórmula π × r²
- Calcula el área de un rectángulo usando la fórmula largo × ancho
- Calcula el área de un triángulo usando la fórmula ½ × base × altura
- Utiliza funciones modulares para cada cálculo

## Uso

Ejecuta el programa con:
```bash
python area_calculator.py
