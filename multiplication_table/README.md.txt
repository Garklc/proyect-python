# Generador de Tablas de Multiplicar y Factorial

Este programa utiliza bucles iterativos para generar tablas de multiplicar y calcular factoriales.

## Características

- Genera tablas de multiplicar del 1 al 10 para cualquier número
- Calcula el factorial de un número dado
- Utiliza bucles `for` y `while`

## Uso

Ejecuta el programa con:
```bash
python multiplication_table.py
