# Determinador de Signo de Número

Este programa utiliza sentencias condicionales para determinar si un número es positivo, negativo o cero.

## Características

- Determina si un número es positivo (> 0)
- Determina si un número es negativo (< 0)
- Determina si un número es cero (= 0)

## Uso

Ejecuta el programa con:
```bash
python number_sign.py
